# CubeKunWars
It is the source code of RTS, CubeKunWars under development.  
Other assets are not included.
# Features
- Unit
- Weapon
- Base point
- Time control
- Random map
# Plans
- Medieval weapons (Sword & Bow)
- Unit customize
- Tutorial
- Animation
- Full-scale implementation of random map
